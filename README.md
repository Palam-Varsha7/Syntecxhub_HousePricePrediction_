# House Price Prediction

## About
A Machine Learning project to predict house prices using Linear Regression.

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- joblib

## How to Run
1. Install libraries: pip install pandas numpy scikit-learn matplotlib seaborn joblib
2. Run: python Syntecxhub_HousePricePrediction.py

## Results
- Model: Linear Regression
- Metrics: RMSE and R² Score
- Model saved as house_price_model.pkl
